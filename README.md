## 📊 Бинарная классификация клиентов

В этом ноутбуке построена модель для задачи бинарной классификации (предсказание целевой переменной `y`).

---

## 🔍 Этапы работы

### 1. Подготовка данных

* преобразование целевой переменной в бинарный формат (`yes → 1`, `no → 0`)
* удаление нерелевантных признаков (`duration`)
* разбиение на train/test с сохранением баланса классов (`stratify=y`)

---

### 2. Бейзлайн модель

* DummyClassifier (частотная стратегия)
* используется как baseline для сравнения
* метрика: ROC-AUC

---

### 3. Логистическая регрессия

* обработка числовых и категориальных признаков:

  * StandardScaler для числовых
  * OneHotEncoder для категориальных
* модель: LogisticRegression
* оценка качества по ROC-AUC

---

### 4. Random Forest + оптимизация гиперпараметров

* используется Optuna для подбора гиперпараметров
* кросс-валидация (StratifiedKFold, 5 фолдов)
* оптимизируются:

  * глубина дерева
  * число деревьев
  * min samples split/leaf
  * max features

---

### 5. Финальная модель

* обучение лучшей конфигурации Random Forest
* финальная оценка на test set
* метрика: ROC-AUC
